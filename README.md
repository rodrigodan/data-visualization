# Data Visualization using D3.js

Toolkit: Python 3.6.7

## Getting Started

### Requirements:

- #### [Python 3.6.7]
- #### [Google Chrome Browser]

## Running the tests

1. Run in your terminal the code bellow:

```
python3 -m http.server
```

2. Open the browser and write in the URL bar:

```
localhost:8000
```
